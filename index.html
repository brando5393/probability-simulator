Welcome to the Probability Simulator

!!!PLEASE BE ADVISED!!!

Please note, running this simulator using the Infinity option may cause devices to overheat and or cause the browser to slow to a crawl and or stop all function requiring that the browser be closed or restarted. This option is available only to show the workload this imposes on a system given its nature, and should only be run for a short time before stopping, thank you.

This simulator is intended to show an example of probability, by setting the value of "i" to a random number between 1 and infinity (or another value) until "i=2"


Please click one of the buttons below to run the simulator.


Select "Infinity" to run the sim with no limit on the range of possible number values that I can contain.
Select "Fixed" to run the sim with a limit of 100,000 on the number range.
